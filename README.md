# SainsPulse

Dashboard & analisis untuk memetakan performa numerasi (Matematika) dan literasi sains (Fisika/Kimia/Biologi) berdasarkan nilai UTS, UAS, dan Nilai.

## Struktur Folder
- data_raw/ : data mentah (tidak diubah)
- data_clean/ : data hasil cleaning
- notebooks/ : notebook cleaning & EDA
- dashboard/ : aplikasi Streamlit
- docs/ : screenshot dokumentasi

## Cara Menjalankan (Local)
1. Install dependencies:
   pip install -r requirements.txt

2. Jalankan dashboard:
   streamlit run dashboard/app.py
